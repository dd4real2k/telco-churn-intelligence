# Telco Churn Intelligence

An end-to-end customer churn analytics project designed to identify at-risk customers, explain churn drivers, and support retention strategy through analytics, machine learning, API deployment, and dashboarding.

## Project Objective
The goal of this project is to help a telecom business:
- understand why customers churn
- predict which customers are likely to churn
- quantify revenue at risk
- simulate targeted retention strategies
